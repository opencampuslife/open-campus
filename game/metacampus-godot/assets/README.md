## 概述

本文档描述 MetaCampus 2D 游戏所需的美术资源。由于无法直接生成图片文件，此处提供详细的**文字描述规范**，供开发者/美术师按照描述制作对应资源。

所有资源遵循 `art-style-guide.md` 中的规格定义：
- Tile 尺寸：32x32 像素
- 角色尺寸：32x32 像素（含 4px 阴影）
- 角色动画：四方向，每方向 3 帧
- UI 分辨率：1280x720

## 五、占位符使用说明

### 5.1 开发阶段占位符

在资源文件创建前，Godot Developer 可以：

1. **使用纯色 PNG**：创建与目标尺寸相同的纯色图片作为占位符
2. **使用临时图标**：从开源图标库（如 Lucide、Font Awesome）临时借用
3. **使用占位符标识**：在资源路径后加 `_placeholder` 便于区分

### 5.2 快速替换流程

当美术资源完成时：
1. 替换对应路径的 PNG 文件
2. Godot 自动检测纹理变化
3. 无需修改代码

### 5.3 推荐开源资源

| 类型 | 来源 | 许可 |
|------|------|------|
| 像素人物模板 | OpenGameArt.org | CC0 |
| 像素瓦片 | Kenney.nl | CC0 |
| UI 图标 | Lucide Icons | ISC |
| UI 图标 | Game Icons | CC BY 3.0 |
