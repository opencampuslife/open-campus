# Phase G2 验收清单

## 概述

G2 目标：让 8 个任务从 JSON 数据变成可执行流程。NPC 触发任务 → 对话选择 → objective 更新 → reward/penalty 生效 → 任务完成 → TaskBoard/Dashboard 展示。
