# 家长择校与陪读指南

## 如何判断孩子是否适合复读

1. 学生本人有强烈复读意愿，而非仅家长意愿
2. 高考成绩与平时成绩有明显差距，有进步空间
3. 学习习惯尚可或有改善意愿
4. 心理状态稳定，能承受复读压力

不建议勉强不愿复读的学生，效果往往不佳。

## 如何选择复读学校

1. 了解学校的教学理念和管理风格是否适合孩子
2. 考察师资水平和教学经验
3. 了解班型设置和学生管理方式
4. 实地查看校园环境、教室和宿舍条件
5. 与在读学生或家长交流了解真实情况

## 陪读注意事项

学校为全日制封闭管理，不建议家长在校区内陪读。家长可以通过以下方式参与：
1. 每周接收学情简报了解孩子情况
2. 定期参加家长会和成绩分析会
3. 通过班主任沟通了解日常表现
4. 协助孩子制定合理目标并给予情感支持

## 家庭配合建议

1. 避免过度施压，关注孩子心理健康
2. 保持合理期望，不要求固定分数目标
3. 创造良好的家庭沟通氛围
4. 配合学校的管理要求和学习安排
