# 异议处理标准话术

## 价格异议

当家长表示价格偏高时：
1. 先了解家长对价格的预期和经济承受范围
2. 介绍费用构成：学费包含教学、管理、测评和学情跟踪
3. 对比市场上的同类全日制复读项目
4. 引导关注教学成果和管理质量而非单纯价格
5. 如有必要，预约到校面谈了解具体优惠政策

不得在电话或线上直接透露具体优惠金额。不得承诺最低价。

## 效果质疑

当家长质疑复读效果时：
1. 分析学生当前情况：基础、弱科、自律性
2. 介绍分层教学和个性化方案如何帮助提升
3. 说明学校管理制度如何保障学习效果
4. 展示教学流程和学情跟踪机制

不得承诺具体提分幅度。不得保证录取结果。

## 距离顾虑

当家长表示距离较远时：
1. 强调全日制封闭管理的优势
2. 介绍住宿条件和安全管理
3. 说明周末放假安排
4. 如有多校区可选，介绍就近校区
